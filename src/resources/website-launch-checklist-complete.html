1. ‚úÖ Pre-Production (D-14)

ESSENTIAL Content & Text

  • All texts are written and approved by client
  • No lorem ipsum or placeholder text on the site
  • Spelling and grammar checked
  • Consistent tone and style across all pages
  • Contact details (email, phone, address) up to date
  • Legal notices written and compliant
  • Privacy policy / GDPR present
  • Terms and Conditions written if e-commerce site

ESSENTIAL Images & Media

  • All final images integrated (no placeholders)
  • Images optimized (WebP, compression, lazy loading)
  • Descriptive ALT text on all images
  • Favicon present and displaying correctly
  • High-resolution logo on all pages
  • Open Graph image for social media sharing (1200√ó630px)
  • Videos hosted on external platform (YouTube, Vimeo) if possible
  • Appropriate image sizes (no 5MB image for 300px display)

ESSENTIAL Features

  • All internal links work (no 404s)
  • All external links open in new tab
  • Forms tested and sending emails correctly
  • Confirmation messages after form submission
  • Clear error messages on form failure
  • Anti-spam protection on forms (reCAPTCHA, honeypot)
  • Site search works (if applicable)
  • Pagination works correctly (if applicable)
  • Filters and sorting work (if applicable)
  • User account: registration, login, logout tested

ESSENTIAL E-commerce (if applicable)

  • All products have price, descriptions, images
  • Product variations (sizes, colors) work
  • Add to cart works
  • Quantity modification in cart works
  • Product removal from cart works
  • Shipping cost calculation correct
  • Complete checkout process tested (real test order)
  • Credit card payment tested in sandbox then live
  • Order confirmation email sent
  • Invoice generated and sent to customer
  • Stock updated after purchase
  • Promo codes work (if applicable)
  • VAT calculation correct

2. üì± Responsive & Compatibility

ESSENTIAL Multi-Device Testing

  • Site tested on iPhone (Safari)
  • Site tested on Android (Chrome)
  • Site tested on iPad tablet
  • Site tested on Android tablet
  • Site tested on Windows desktop (Chrome, Firefox, Edge)
  • Site tested on Mac desktop (Chrome, Safari, Firefox)
  • Keyboard navigation possible (accessibility)
  • Buttons and links large enough for mobile (min 44√ó44px)
  • Text readable without zoom on mobile (min 16px)
  • No horizontal scroll on any device

IMPORTANT Breakpoints & Display

  • Breakpoints tested: 375px (mobile S), 768px (tablet), 1024px (desktop)
  • Responsive images adapted to each screen size
  • Functional hamburger menu on mobile
  • Forms usable on mobile (proper spacing)
  • Popups and modals display correctly on mobile
  • Footer readable and well-structured on mobile

3. üöÄ Performance & Speed

ESSENTIAL Technical Optimizations

  • Lighthouse score > 90 on mobile
  • Lighthouse score > 90 on desktop
  • Loading time < 3 seconds (4G)
  • First Contentful Paint < 1.8s
  • Largest Contentful Paint < 2.5s
  • Images optimized and compressed
  • WebP or AVIF format used for images
  • Lazy loading enabled on images outside viewport
  • CSS and JS minified
  • Unused files removed

IMPORTANT Cache & CDN

  • Browser cache configured (Cache-Control headers)
  • CDN enabled if hosting allows (Cloudflare, etc.)
  • Gzip/Brotli compression enabled
  • HTTP/2 or HTTP/3 enabled on server
  • Preconnect and DNS-prefetch for external resources
